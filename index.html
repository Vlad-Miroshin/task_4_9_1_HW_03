При выполнении работы были соблюдены следующие требования:

  • Сайт содержит три страницы: «Главную», «Обо мне» и «Контакты».
  • На всех страницах есть шапка, основное содержимое и подвал. Использованы семантические теги.
  • Все страницы связаны.
  • Заполнена главная веб-страница index.html.
  • Заполнена страница about.html.
  • Заполнена страница contacts.html.
  • Ссылки на страницы работают (в том числе локально, то есть запускаются из папки проекта), по ним осуществляется переход.
  • Текстовое содержимое заключено в теги.
  • Добавлены изображения тегом <img>.
  • Страницы сайта проверены через Validator W3C и не имеют ошибок.
  • Страницы содержат ссылки на внешние ресурсы. Использованы спецсимволы html.
  • Файлы сайта размещены в публичном репозитории на GitHub.
